# JavaScript在线编辑器

一个功能强大的在线JavaScript代码编辑器，支持HTML、CSS、JavaScript和D3.js实时编辑和预览。

## 功能特色

### 🎯 核心功能
- **多标签编辑器**: 支持HTML、CSS、JavaScript和D3.js四种代码编辑
- **实时预览**: 代码修改后自动更新预览效果
- **语法高亮**: 使用CodeMirror提供专业的代码编辑体验
- **智能提示**: 支持代码自动补全和语法检查
- **控制台输出**: 实时显示console.log输出和错误信息
- **本地存储**: 自动保存代码，刷新页面不会丢失

### 📊 D3.js数据可视化
- **5种内置示例**: 柱状图、饼图、折线图、散点图、力导向图
- **交互式图表**: 支持鼠标悬停、拖拽等交互功能
- **动画效果**: 流畅的图表生成动画
- **完整D3.js支持**: 可编写自定义D3.js可视化代码

### 🛠️ 用户体验
- **现代化界面**: 深色主题，专业的开发环境
- **快捷键支持**: Ctrl+S保存，Ctrl+Enter运行代码
- **全屏预览**: 支持全屏查看预览效果
- **代码导出**: 一键导出为HTML文件
- **响应式设计**: 适配不同屏幕尺寸

## 使用方法

1. **打开编辑器**: 双击`index.html`在浏览器中打开
2. **编写代码**: 在不同标签页中编写HTML、CSS、JavaScript代码
3. **选择D3示例**: 在D3.js标签页中选择预设的可视化示例
4. **实时预览**: 右侧预览区域会实时显示代码效果
5. **保存和导出**: 使用顶部工具栏保存代码或导出项目

## 技术栈

- **前端框架**: 纯HTML/CSS/JavaScript（无框架依赖）
- **代码编辑器**: CodeMirror 5.65.15
- **数据可视化**: D3.js v7
- **样式设计**: 自定义CSS + Font Awesome图标
- **浏览器兼容**: 支持所有现代浏览器

## 快捷键

- `Ctrl + S`: 保存代码
- `Ctrl + Enter`: 运行代码
- `Ctrl + Space`: 显示代码提示
- `F11`: 切换全屏编辑模式

## 功能按钮

- **运行代码**: 执行当前代码并更新预览
- **清空**: 清空所有编辑器中的代码
- **保存**: 保存代码到浏览器本地存储
- **导出**: 导出当前代码为HTML文件
- **刷新预览**: 重新运行代码并刷新预览
- **全屏**: 全屏查看预览效果

## D3.js示例

编辑器内置了5个D3.js可视化示例：

1. **柱状图**: 展示不同编程语言的熟练程度
2. **饼图**: 显示技能分布情况
3. **折线图**: 展示销售数据趋势
4. **散点图**: 显示数据点分布
5. **力导向图**: 展示节点关系网络

## 开发说明

### 文件说明

- `index.html`: 主页面，包含编辑器界面结构
- `style/main.css`: 样式文件，定义编辑器外观
- `js/editor.js`: 核心功能实现，包含编辑器逻辑
- `js/d3-examples.js`: D3.js示例代码库

### 主要类和方法

- `JSEditor`: 主编辑器类
  - `init()`: 初始化编辑器
  - `setupEditors()`: 设置CodeMirror编辑器
  - `runCode()`: 执行代码并更新预览
  - `saveCode()`: 保存代码到本地存储
  - `exportCode()`: 导出代码为HTML文件
